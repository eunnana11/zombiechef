# 좀비 요리사

React, TypeScript, Vite, Tailwind CSS로 만든 2~4인 로컬 턴제 전략 보드게임입니다.

플레이어는 좀비가 된 셰프입니다. 공용 포인트 카드 덱을 밀당하며 뽑고, 보드를 돌며 식재료를 모으고, 주방에서 요리를 완성해 가장 높은 점수를 노립니다.

## 실행 방법

가장 쉬운 방법:

```text
start-game.cmd 더블클릭
```

PowerShell을 쓰는 방법:

```bash
npm install
npm run dev
```

브라우저에서 Vite가 안내하는 로컬 주소를 열면 플레이할 수 있습니다.

## 게임 목표

- 15라운드가 끝나면 게임 종료
- 또는 누군가 요리 4개를 완성하면 그 라운드 끝까지 진행 후 종료
- 또는 누군가 한 바퀴를 돌아 시작점을 다시 밟거나 지나치면 즉시 종료
- 최종 점수가 가장 높은 플레이어가 승리
- 동점이면 완성 요리 수, 남은 식재료 수 순서로 순위 결정

## 턴 흐름

1. 현재 플레이어가 공용 포인트 카드 덱에서 카드를 1장 뽑습니다.
2. 원하면 최대 3장까지 추가로 뽑을 수 있습니다.
3. 뽑기를 멈추면 최종 포인트를 계산합니다.
4. 최종 포인트가 음수면 그만큼 자동 후진하고 턴이 종료됩니다.
5. 최종 포인트가 0이면 자동 턴 종료됩니다.
6. 최종 포인트가 양수면 포인트가 남아있는 동안 여러 행동을 할 수 있습니다.
7. 포인트가 0이 되면 자동 턴 종료됩니다.
8. 모든 플레이어가 1턴씩 마치면 라운드 종료입니다.
9. 라운드 종료 후 남은 덱과 버린 더미를 모두 섞어 새 공용 덱을 만듭니다.

## 포인트 카드 덱

모든 플레이어가 하나의 공용 덱을 사용합니다.

- `+1` 카드 10장
- `0` 카드 4장
- `-1` 카드 6장
- `+3` 카드 7장
- `-3` 카드 3장
- 총 30장

턴에 뽑은 카드는 턴 종료 시 버린 더미로 갑니다.

## 특수 조합

기본은 카드 숫자를 그대로 더합니다. 단, 아래 조합이면 특수 규칙을 적용합니다.

- 마이너스 카드 2장: 최종 `-1P`
- 마이너스 카드 3장: 최종 `+4P`
- 마이너스-마이너스-플러스 순서: 최종 `-4P`
- 플러스 카드 3장: 기본 합산값 `+4P`
- 0 카드 2장: 최종 `+1P`
- 0 카드 3장: 최종 `+3P`

마이너스-마이너스-플러스는 순서가 중요합니다. 예를 들어 `-1, -3, +1`은 특수 조합이지만 `-1, +1, -3`은 아닙니다.

특수 조합이 발동하면 중앙 팝업과 게임 로그에 전용 문구가 표시됩니다.

- 마이너스 카드 2장: "좀비 근성 발동! 간신히 버텼다!"
- 마이너스 카드 3장: "완전한 좀비 각성! 오히려 강해졌다!"
- 마이너스-마이너스-플러스: "그냥 멈췄어야 했다..."
- 플러스 카드 3장: "셰프의 황금 타이밍!"
- 0 카드 2장: "신중한 준비 완료."
- 0 카드 3장: "완벽한 준비가 끝났다!"

## 행동 포인트

최종 포인트는 행동 포인트로 사용합니다.

- 이동: `1P = 1칸`
- 팔 구매: 팔마다 비용 지불
- 요리 제작: 주방 칸에서 요리 비용 지불

포인트가 남아 있으면 여러 행동을 이어서 할 수 있습니다. 예를 들어 `4P`가 있으면 `2P로 2칸 이동` 후 `2P 팔 구매`가 가능합니다.

## 보드

현재 보드는 30칸 사각 트랙입니다.

- 시작 1칸
- 식재료 칸 10칸
- 탐색 칸 9칸
- 주방 2칸
- 최종 주방 1칸
- 빈칸 7칸

식재료와 탐색 칸이 많아서 매 턴 재료 수급이나 이벤트가 자주 발생합니다.

29번 칸을 지나 앞으로 더 이동하면 0번 시작점으로 이어집니다. 한 플레이어라도 시작점을 다시 밟거나 지나치면 게임이 즉시 종료됩니다.

## 식재료

식재료는 5종입니다.

- 고기 `🍖`
- 버섯 `🍄`
- 허브 `🌿`
- 생선 `🐟`
- 곡물 `🌾`

식재료 칸에 도착하면 기본으로 랜덤 식재료 2개를 얻습니다. 집게팔이 있으면 1개를 추가로 얻습니다.

기본 휴대량은 5개입니다. 휴대량을 초과하면 폐기 선택 모달이 뜨고, 플레이어가 초과 개수만큼 직접 식재료를 골라 버립니다. 선택 완료 전에는 다른 행동을 할 수 없습니다.

## 탐색 카드

탐색 칸에 도착하면 탐색 카드 효과가 발동하고, 결과가 팝업과 탐색 로그에 표시됩니다.

현재 탐색 효과:

- 앞으로 2칸 이동
- 뒤로 2칸 이동
- 랜덤 식재료 1개 획득
- 식재료 1개 잃음
- 다음 턴 스킵
- 다음 요리 비용 -1
- 다른 플레이어 1칸 밀기
- 버섯 1개 획득
- 다음 팔 구매 비용 -1
- 가장 가까운 주방 방향 2칸 이동
- 랜덤 식재료 2개 획득
- 생선 전부 잃음
- 다른 플레이어와 위치 교환
- 다른 플레이어 식재료 1개 훔치기
- 곡물 1개 획득
- 고기 1개 획득
- 가장 가까운 식재료 칸으로 이동
- 이번 턴 이동 +1
- 즉시 2칸 후진
- 허브 1개 획득

멀티플레이 상호작용은 탐색 카드에 들어 있습니다. 밀기, 위치 교환, 재료 훔치기 같은 효과가 다른 플레이어를 견제합니다.

## 팔 카드

플레이어는 최대 2개의 팔을 장착할 수 있습니다.

팔 슬롯이 비어 있으면 구매한 팔을 바로 장착합니다. 이미 팔이 2개 장착된 상태에서 새 팔을 구매하려면, 기존 팔 1개를 선택해 버리고 새 팔로 교체할 수 있습니다. 교체 추가 비용은 없고 새 팔의 구매 비용만 지불합니다.

팔 교체로 바구니팔이나 수레팔을 잃어 최대 휴대량이 줄어들 수 있습니다. 이때 보유 식재료가 최대 휴대량을 초과하면 폐기 선택 모달이 뜨고, 폐기된 식재료와 개수는 게임 로그에 기록됩니다.

- 집게팔 `1P`: 식재료 칸에서 획득 +1
- 바구니팔 `2P`: 휴대량 +3
- 도끼팔 `2P`: 부정적 탐색 이벤트 1회 무시
- 국자팔 `2P`: 요리 포인트 비용 -1
- 낚싯대팔 `1P`: 생선 획득 시 +1
- 갈고리팔 `2P`: 탐색 칸에서 식재료를 얻으면 +1개
- 황금 셰프팔 `3P`: 완성 요리마다 +2점
- 수레팔 `3P`: 휴대량 +5

## 요리

주방 또는 최종 주방 칸에서 요리를 만들 수 있습니다.

요리에는 필요한 식재료, 포인트 비용, 점수가 있습니다. 포인트 비용은 전반적으로 낮게 잡혀 있어 이동 후에도 요리를 노릴 수 있습니다.

대표 요리:

- 좀비 스테이크
- 버섯 수프
- 생선 파이
- 뇌 모양 샐러드
- 황금 좀비 정식
- 셰프의 최후의 만찬

## 현재 밸런스 방향

이 게임은 카드 운으로 행동 포인트를 얻고, 이동은 `1P = 1칸`으로 처리합니다. 함정 시스템은 제거했고, 대신 탐색 카드의 견제 효과로 플레이어 간 상호작용을 만들었습니다.
